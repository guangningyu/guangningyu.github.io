Newton's Method

In numerical analysis, Newton’s method (also know as the Newton-Raphson method), named after Isaac Newton and Joseph Raphson, is a method for finding successively better approximations to the roots (or zeroes) of a real-valued function.

Description

Finding $$ \sqrt{S} $$ is the same as solving the equation $$ f(x) = x^2 - S = 0 $$

The idea of the method is as follows: one starts with an initial guess which is reasonably close to the true root, then the function is approximately by its tangent line, and one computes the x-intercept of this tangent line. This x-intercept will typically be a better approximation to the function’s root than the original guess, and the method can be iterated.

Newton’s method reduces in this case to the so-called Babylonian method:
$$ x_{n+1} = x_n - \dfrac{f(x_n)}{f’(x_n)} = x_n - \dfrac{x_n^2 - S}{2x_n} = \dfrac{1}{2}(x_n + \dfrac{S}{x_n}) $$

Generally, these methods yield approximate results. To get a higher precision for the root, a higher precision for the square is required and a large number of steps must be calculated.

Proof of Quadratic Convergence

According to Taylor’s theorem, any function f(x) which has a continuous second derivative can be represented by an expansion about a point that is close to a root of f(x). Suppose this root is α. Then the expansion of f(α) is:

$$ f(α) = f(x_n) + f’(x_n)(α-x_n)+R_1\ \ \ \ \ (1)$$

where the Lagrange form of the Taylor series expansion remainder is

$$ R_1 = \dfrac{1}{2!}f’’(ξ_n)(α-x_n)^2,\ where\ ξ_n\ is\ in\ between\ x_n\ and\ α.$$

Since α is the root, (1) becomes:
$$0 = f(α) = f(x_n) + f’(x_n)(α-x_n) + \dfrac{1}{2!}f’’(ξ_n)(α-x_n)^2\ \ \ \ \ (2)$$

Then it gives,

$$\dfrac{f(x_n)}{f’(x_n)}+(α-x_n)=\dfrac{-f’’(ξ_n)}{2f’(x_n)}(α-x_n)^2\ \ \ \ \ (3)$$

Remembering that

$$x_{n+1} = x_n - \dfrac{f(x_n)}{f’(x_n)}\ \ \ \ \ (4)$$

Then

$$ α-x_{n+1} = \dfrac{-f’’(ξ_n)}{2f’(x_n)}(α-x_n)^2 $$

Let

$$ ε_n = α-x_n $$

Then we have

$$ ε_{n+1} = \dfrac{-f’’(ξ_n)}{2f’(x_n)}ε_n^2 \ \ \ \ \ (5) $$

Taking absolute value of both sides gives

$$ |ε_{n+1}| = \dfrac{|f’’(ξ_n)|}{2|f’(x_n)|}ε_n^2 \ \ \ \ \ (6) $$

Equation (6) shows that the rate of convergence is quadratic if the following conditions are satisfied:

$$ 1.\ f’(x) ≠ 0;\ ∀x∈I, where\ I\ is\ the\ interval\ [α-r, α+r]\ for\ some\ r≥|(α-x_0)|; $$
$$ 2.\ f’’(x)\ is\ finite,\ ∀x∈I; $$
$$ 3.\ x_0\ sufficiently\ close\ to\ the\ root\ α $$

The term sufficiently close in this context means the following:
$$ (a)\ Taylor\ approximation\ is\ accurate\ enough\ such\ that\ we\ can\ ignore\ higher\ order\ terms $$

$$ (b)\ \dfrac{1}{2}|\dfrac{f’’(x_n)}{f’(x_n)}| < C|\dfrac{f’’(α)}{f’(α)}|,\ for\ some\ C < ∞ $$

$$ (c)\ C|\dfrac{f’’(α)}{f’(α)}|ε_n < 1,\ for\ n∈Z^+∪\{0\}\ and\ C\ satisfying\ condition\ (b) $$

Finally, (6) can be expressed in the following way:

$$ |ε_{n+1}| ≤ Mε_n^2 $$

where M is the supremum of the variable coefficient on the interval I defined in the condition 1, that is:

$$ M = \sup\limits_{x∈I}\dfrac{1}{2} |\dfrac{f’’(x)}{f’(x)}|$$

The initial point has to be chosen such that condition 1 through condition 3 are satisfied, where the third condition requires that

$$ M|ε_0| < 1 $$

In some cases the conditions on the function that are necessary for convergence are satisfied, but the point chosen as the initial point is not in the interval where the method converges. This can happen, for example, if the function whose root is sought approaches zero asymptotically as x goes to ∞ or -∞. In such cases a different method, such as bisection, should be used to obtain a better estimate for the zero to use as an initial point.

Sample Code

def square_root(n):
    iteration = 20   # times of iteration
    root = n / 2.0   # first guess

    for k in range(iteration):
        root = (1.0 / 2.0) * (root + (n / root))
    return root
